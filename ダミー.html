人とパソコンは言語が違う

パソコンが理解できるデータをバイナリーデータと言う

プログラムは翻訳され実行される

プログラムを翻訳する操作をコンパイルとよぶ

コンパイルの扱いは言語ごとに違う

  • コンパイル言語（Go、C++、Java）
    一度コンパイルしたファイルを使い回す

  • インタプリタ言語（Python、R、Javascript）
    実行するたびにコンパイルする

インストールってなに？

ソフトウェアをパソコンへ導入し、使用可能な状態にする処理や作業のこと

ソフトウェアのインストールには大きくわけて2種の方法がある

  • ソースコードからソフトウェアを構築する
    （ソースからビルド）
  • パッケージマネージャーを使用する
    （Homebrew,condaなど）

ソースコードからのインストール

コンパイル言語の場合

  1. ソースコードのダウンロード
  2. コンパイル用のファイルを作成（ない場合もある）
  3. コンパイル実行
  4. 実行ファイルがある場所へパスを通す

インタプリタ言語の場合

  1. ソースコードのダウンロード
  2. ダウンロードした場所へパスを通す

コンパイル言語インストール体験

  1. ソースコードのダウンロード
git clone https://github.com/YutaSSuzuki/install_practice.git
  1. コンパイル用のファイル（Makefile）を作成：
cd compile
./configure
  1. コンパイル実行（実行ファイルを作成）:
make

コンパイル言語インストール体験

  1. パスを通す:
pwd #出力結果をコピー
nano ~/zsh_profile #shellの種類に応じて_の前を変更

# zsh_profile内の一番下の行に以下の内容を追記
PATH="コピー内容をペースト:$PATH"

プログラムの実行

prime

🔰 好きな整数を半角で入力し、素数判定を行う
🔰 ディレクトリを移動しprimeを実行しよう

インタプリタ言語インストール体験

  1. pythonコードのダウンロード
git clone
  1. パスを通す
cd interpreter/
pwd #出力結果をコピー
nano ~/zsh_profile #shellの種類に応じて_の前を変更

# zsh_profile内の一番下の行に以下の内容を追記
PATH="コピー内容をペースト:$PATH"

プログラムの実行

prime.py

🔰 好きな整数を半角で入力し、素数判定を行う
🔰 ディレクトリを移動しprime.pyを実行しよう

余談：新しいコンパイルcmake

これまではconfigureで環境を読み取り、Makefileを作るのが一般的だった。
しかし最近はcmakeを活用したMakefileの作成が主流

cmakeの特徴として以下のものが挙げられる

  • 高速
  • 作成側とインストール側の両方でcmakeのインストールが必要

パッケージマネージャーとは？

ソフトウェアのインストールやバージョン管理を行う
ソフトのこと
（Homebrew,Anaconda,pip3など）

以下の利点がある

  • インストールをコマンド１つで行える
  • アップデートの際、ソフトを入れ直す必要がない
  • インストールに必要なものを全て入れてくれる

Homebrewって何？

MacOS、Linux用のパッケージマネージャー

Github上に公式が公開しているリポジトリから
ソフトウェアをインストールできる。

  • インストール
brew install <ソフト名>
  • インストールしたソフトの一覧表示
brew list
  • Homebrewで入れたソフトのアップデート
brew update

便利機能

bottle

バイナリーデータを直接ダウンロードする。

以下の条件下で使用できなくなるので注意

  • MacOSのバージョンが対応していない
  • Homebrewがデフォルトの場所にインストールされてい

tap

Github上にアップロードされている公式以外の
リポジトリのソフトをインストールできる。

brew install <github_user_name>/tap/<soft_name>

インストール時のエラー対応

  • パッケージマネージャーを使用
  • バイナリーデータを直接ダウンロード
  • ソフトウェアのissueを確認

解決が難しい状況

  • 管理者権限がない

  • 必須のソフトの配布が終了している

対応策

  • Apptainerなど構築済みのものをパスを指定して使用する
  • 別のソフトウェアを探す
  • 環境を変える
    • DDBJからScorpionへ移動する
